# Plan de Implementacion Cliente (Operacion Real)

## Paso 1 - Preinstalacion
- Validar hardware caja principal y red local.
- Instalar dependencias (XAMPP/Node) segun guia tecnica.
- Validar impresora ticket y perifericos.

## Paso 2 - Instalacion
- Desplegar version estable.
- Configurar base de datos.
- Levantar servidor Node y sitio local.
- Crear usuario administrador.

## Paso 3 - Parametrizacion
- Cargar datos del negocio.
- Asignar cajas por dispositivo.
- Configurar medios de pago, impuestos, moneda y ticket.
- Cargar catalogo inicial de productos.

## Paso 4 - Pruebas en sitio
- Ejecutar matriz QA minima de ventas/corte/inventario.
- Simular cierre de turno completo.
- Validar reportes y exportaciones.

## Paso 5 - Puesta en marcha
- Capacitar cajero y encargado.
- Definir canal de soporte.
- Entregar manuales y SLA.
- Acta de aceptacion operativa.
