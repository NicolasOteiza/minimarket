# Checklist de Salida Comercial

## A. Cierre Tecnico (obligatorio)
- [ ] Sesion/turno validado sin perdida de datos en turnos largos.
- [ ] Flujo multi-caja validado (hasta 8 equipos).
- [ ] Sincronizacion de logo/configuracion entre equipos.
- [ ] Fallback ante caida temporal de backend (lectura local controlada).
- [ ] Manejo de errores con registro y notificacion implementado.
- [ ] Exportaciones (CSV/XLSX) validadas en Excel real.

## B. QA Funcional
- [ ] Ventas: escaneo, busqueda, cobro, reimpresion.
- [ ] Productos: crear, modificar, eliminar con reglas de inventario.
- [ ] Inventario: reposicion y validaciones.
- [ ] Compras: orden, envio, recepcion y cierre.
- [ ] Corte: resumen correcto por sesion y por dia.
- [ ] Reportes: graficos y descargables consistentes.

## C. Operacion
- [ ] Manual operador (caja/encargado).
- [ ] Manual administrador.
- [ ] Procedimiento de respaldo y restauracion probado.
- [ ] Procedimiento de actualizacion documentado.

## D. Comercial/Legal
- [ ] Modelos comerciales definidos (local + SaaS).
- [ ] Planes y precios publicados internamente.
- [ ] SLA acordado y documentado.
- [ ] Contrato base listo para firma.

## E. Go-Live
- [ ] Piloto con cliente real.
- [ ] Correccion post-piloto.
- [ ] Version estable etiquetada.
