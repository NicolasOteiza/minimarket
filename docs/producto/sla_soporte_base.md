# SLA de Soporte Base

## Alcance
Aplica a incidencias del sistema en ambiente productivo.

## Niveles de severidad
- Critica (S1): sistema caido o venta bloqueada general.
- Alta (S2): funcionalidad principal degradada con workaround parcial.
- Media (S3): falla no bloqueante.
- Baja (S4): dudas, mejoras menores, ajustes cosmeticos.

## Tiempos objetivo
- S1: respuesta <= 2 horas habiles, contencion <= 8 horas.
- S2: respuesta <= 4 horas habiles, contencion <= 24 horas.
- S3: respuesta <= 1 dia habil, solucion en backlog acordado.
- S4: respuesta <= 2 dias habiles.

## Horario base
- Lunes a viernes, 09:00 a 18:00 (Chile).

## Canales
- Correo de soporte.
- WhatsApp empresarial (segun contrato).

## Exclusiones
- Falla de internet del cliente.
- Falla electrica o hardware del cliente.
- Modificaciones no autorizadas a BD o codigo.
