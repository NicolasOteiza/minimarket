# Checklist Operativo por Rol

## Cajero
- [ ] Inicia sesión correctamente.
- [ ] Escanea producto y lo agrega al carrito.
- [ ] Busca producto por nombre.
- [ ] Ajusta cantidad y elimina producto seleccionado.
- [ ] Cobra venta (con y sin comprobante).
- [ ] Reimprime última venta.
- [ ] Cierra turno con datos declarados.

## Encargado de inventario
- [ ] Carga producto en inventario por código o nombre.
- [ ] Registra reposición.
- [ ] Confirma actualización de stock.
- [ ] Verifica productos sin inventario activo.

## Encargado de compras
- [ ] Crea orden de compra.
- [ ] Agrega productos a la orden.
- [ ] Envía orden por correo.
- [ ] Recibe pedido y marca cantidades recibidas.
- [ ] Cierra recepción con faltantes/ajustes.

## Administrador
- [ ] Configura cajas y cajeros.
- [ ] Revisa reportes y exportaciones.
- [ ] Valida corte por caja y por día.
- [ ] Configura correo/notificaciones.
- [ ] Verifica logo/configuración en múltiples equipos.
