# Planes y Tarifas Base (Referencia Comercial)

> Documento base editable. Valores en CLP.

## Plan 1 - Local Pago Unico
- Implementacion inicial: CLP 590.000
- Licencia perpetua 1 local: incluida
- Cajas incluidas: hasta 2
- Soporte correctivo: 3 meses
- Capacitacion inicial: 1 jornada remota

### Adicionales
- Caja extra local: CLP 60.000
- Soporte mensual post-garantia: CLP 39.900
- Desarrollo a medida: cotizacion

## Plan 2 - SaaS Mensual
- Setup inicial: CLP 120.000
- Mensualidad base: CLP 49.900 / local
- Cajas incluidas: hasta 2
- Soporte: incluido segun SLA base
- Respaldo: incluido

### Adicionales
- Caja extra: CLP 9.900 / mes
- Multi-sucursal: cotizacion
- Reporteria avanzada: cotizacion

## Condiciones Comerciales Base
- Forma de pago: transferencia.
- Vigencia de cotizacion: 10 dias corridos.
- No incluye hardware.
- Facturacion electronica: se habilita cuando cliente entrega credenciales y certificados requeridos.
