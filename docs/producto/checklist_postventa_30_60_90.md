# Checklist Post-Venta (30/60/90 días)

## Día 30
- [ ] Revisión de operación diaria (ventas/corte).
- [ ] Validación de respaldos.
- [ ] Revisión de incidentes reportados.
- [ ] Ajustes menores de configuración.
- [ ] Capacitación de refuerzo (si aplica).

## Día 60
- [ ] Auditoría de calidad de datos (productos/inventario).
- [ ] Revisión de flujo de compras y recepción.
- [ ] Validación de reportes de gestión.
- [ ] Revisión de usuarios/permisos.
- [ ] Estado de soporte y SLA.

## Día 90
- [ ] Evaluación de continuidad operativa.
- [ ] Revisión comercial de renovación/upgrade.
- [ ] Identificación de mejoras de alto impacto.
- [ ] Cierre de etapa inicial de adopción.

## Indicadores sugeridos
- Tiempo promedio de venta.
- Diferencia de caja por cierre.
- Tasa de incidencias críticas.
- Cumplimiento de respaldo.
