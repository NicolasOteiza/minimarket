# Roadmap de Cierre de Producto

## Fase 1 - Endurecimiento tecnico (Semana 1-2)
1. Estabilidad de sesiones y turnos.
2. Validacion multi-dispositivo (Windows/Mac/iPad/Android).
3. Correccion final de layout responsive en ventas.
4. Normalizacion de rutas API por equipo.

## Fase 2 - QA formal (Semana 2-3)
1. Matriz de casos de prueba por modulo.
2. Ejecucion de pruebas en ambiente local.
3. Registro de defectos y cierre.

## Fase 3 - Operacion y soporte (Semana 3-4)
1. Manuales de uso.
2. Procedimientos de respaldo/restauracion.
3. Protocolo de soporte y escalamiento.

## Fase 4 - Comercializacion (Semana 4)
1. Planes y tarifas finales.
2. Contrato y SLA definitivos.
3. Piloto con cliente objetivo.
4. Lanzamiento controlado.
