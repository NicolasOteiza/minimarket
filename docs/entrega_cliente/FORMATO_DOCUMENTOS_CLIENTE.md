# Formato Unificado de Documentos Cliente

## Convención de documento
1. Portada breve (título + fecha + contacto).
2. Objetivo del documento.
3. Contenido en bloques numerados.
4. Sección final de observaciones/firma (si aplica).

## Estilo recomendado para PDF
- Tipografía: Arial o Calibri.
- Título principal: 18-20 pt.
- Subtítulos: 13-14 pt.
- Texto: 11 pt.
- Interlineado: 1.15.
- Márgenes: 2 cm.

## Estructura de nombres de archivo
- `NOMBRE_DOCUMENTO_v1_YYYYMMDD.pdf`
- Ejemplo: `Propuesta_Comercial_SIA_v1_20260307.pdf`

## Control de versiones
- Versión inicial: v1.
- Cambios menores: v1.1, v1.2...
- Cambios mayores: v2, v3...

## Revisión mínima antes de enviar
- [ ] Fecha y datos cliente correctos.
- [ ] Valores comerciales actualizados.
- [ ] Alcance y exclusiones visibles.
- [ ] Contacto de soporte correcto.
