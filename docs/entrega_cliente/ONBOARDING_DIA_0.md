# Onboarding Día 0 (Cliente Nuevo)

## Objetivo
Dejar el sistema operativo el primer día con flujo mínimo completo.

## Bloque 1 - Preparación (30 min)
- Confirmar modalidad (Local o SaaS).
- Confirmar equipo principal y periféricos.
- Confirmar usuario administrador.

## Bloque 2 - Configuración inicial (45 min)
- Registrar datos del negocio.
- Asignar caja al dispositivo.
- Configurar impresora ticket.
- Configurar medios de pago.

## Bloque 3 - Carga base (45 min)
- Crear departamentos principales.
- Cargar productos críticos.
- Definir proveedores/encargados.

## Bloque 4 - Prueba operativa (30 min)
- Realizar venta de prueba.
- Imprimir ticket.
- Ejecutar cierre de turno de prueba.
- Revisar reporte básico.

## Bloque 5 - Cierre del día (15 min)
- Validar funcionamiento con responsable del local.
- Registrar pendientes menores.
- Definir fecha de seguimiento Día 30.
